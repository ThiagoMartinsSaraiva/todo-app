## Requirements

- [Node](https://nodejs.org/en/)

## Getting Started

Step by step to run this app in your local

### Install packages

###### Only Project TS

```
cd ./Project TS
```

At your directory root, run:

```
npm install

our

npm install --force
```

### Run server

```
npm start
```

Open on web: http://localhost:1234
